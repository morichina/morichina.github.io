算法和思路

计算 1 + 2 + … + n的计算方法主要有三种：平均计算，迭代，递归

方法一：平均计算

问题：此计算必须使用乘除法，不可取

方法二：迭代

问题：此计算需用到 for，while 等关键字，不可取

方法三：递归

问题：终止递归条件需要使用 if 或 switch等判断语句，不可取

思考：除了 if 或 switch等判断语句，能不能用其他方法来终止递归
